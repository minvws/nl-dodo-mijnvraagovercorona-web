import { UpdatedAt } from '@lib/elements/UpdatedAt';

The `UpdatedAt` component is used to indicate the time and date of the latest content update.

## Usage
Pass in a ISO string in the `time` variable for example `2023-11-29T15:19:35Z`. For translation of the date you need to pass in a country code such as `nl`, `en` (or other) in the `locale` variable.

```jsx
import { UpdatedAt } from '@mvoc/ui/elements';

<UpdatedAt time={'2023-11-29T15:19:35Z'} locale={'en'}>Updated at: </UpdatedAt>
```

<div class="not-content">
