The `Modal` component is used to display content in a overlay and is used to display content that is not part of the main flow of the page.

## Usage

```jsx
import { Modal } from '@dodo/ui/components';

<button type="button" id="modal-ID">Open Modal</button>

<Modal id="modal-ID" title="Modal title">
	Modal content
</Modal>
```

## Focus

By default the modal will focus on the first focusable element inside the modal. If there is no focusable element, the modal will focus on the modal title.

<div class="not-content">
	<Button type="button" id="modal-search">Open search modal</Button>

	<Modal id="modal-search" title="Modal title">
		<InputText
			type={'search'}
			id={'search-field'}
			placeholder={'Zoek op plaatsnaam...'}
			icon={'looking-glass'}
		/>
	</Modal>
</div>

```jsx
<Button type="button" id="modal-search">Open search modal</Button>
```
