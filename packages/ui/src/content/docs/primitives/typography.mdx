The `typography` primitive is a set of font sizes and line heights used for
headings and text. The lineheights are based on our baseline grid of `4px`.

## Usage

Give the `font-size` and `line-height` css property a `--font-size-[size]`, and `--line-height-[size]`
value, where `[size]` equals `'nano','milli', 'delta', 'gamma','beta',` or `'alpha',`
