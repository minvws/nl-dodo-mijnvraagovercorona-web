import { FieldDatePicker } from '@lib/components/Form';

FieldDatePicker is a component that renders a date picker field.
You can provide additional content as a fallback for the date picker.

```jsx
import { FieldDatePicker } from '@dodo/ui/components';

<FieldDatePicker label="Field label" id="unique-id">[optional alternative to the datepicker]</FieldDatePicker>
```
