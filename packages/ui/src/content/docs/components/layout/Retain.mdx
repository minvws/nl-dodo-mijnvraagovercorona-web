`Retain` is a component that can create a horizontally centered column. With such a component, we could create a centered 'stripe' of content within any container, capping its width to preserve a reasonable measure.

## Usage

```jsx
import { Retain } from '@dodo/ui/components';

<Retain>
	// content
</Retain>
```

## Default
By default, `Retain` will create a column that is `696px` (converted to `rem`) wide and is padded on the inline-axis. On larger screens, it will be `1264px` (converted to `rem`) wide.

<div class="not-content">
	<Retain>
		<p>Lorem ipsum dolor sit amet consectetur adipisicing elit. Perferendis est voluptate deleniti atque molestiae, consectetur temporibus labore autem illo sequi dolorum veniam et voluptatem distinctio numquam perspiciatis eum iste! Quo?</p>
	</Retain>
</div>

```jsx
<Retain>
	// content
</Retain>
```

## Custom properties

The following custom properties are available on `Retain`:

```scss
/**
 * 1. The maximum inline-size of the column, defaults to `696px` (converted to `rem`).
 * 2. The padding on the inline-axis, defaults to `var(--space-4)`.
 * 3. The padding on the inline-axis at the start of the column, defaults to `var(--retain-padding)`.
 * 4. The padding on the inline-axis at the end of the column.
 */
.your-component-extending-retain {
	--retain-size: 65ch; /* [1] */
	--retain-padding: var(--space-2); /* [2] */
	--retain-padding-start: var(--space-3); /* [3] */
	--retain-padding-end: var(--space-5); /* [4] */
}
```
