`Links` are used for navigating. The base style is used in body text and general links to other pages. Additionaly a `Link` can have an icon which by default is placed after the label. With configurable options you can place the icon in front of the label. You can also pass the `variant` variable to style the `Link` to default presets.

## Usage

```jsx
import { Link } from '@mvoc/ui/elements';

<Link href="#">Link label</Link>
```

## Base Link
<div class="not-content">
	Body text with a <Link href="#">link</Link> in it.
</div>

```jsx
<p>Body text with a <Link href="#">link</Link> in it.</p>
```
