import { Switcher } from '@lib/components/Layout/Grid';

The `Switcher` component switches between a horizontal and a vertical layout at a given, container-based breakpoint.

## Usage

```jsx
import { Switcher } from '@dodo/ui/components';

<Switcher>
	<div>
		column 1
	</div>
	<div>
		column 2
	</div>
</Switcher>
```

## Default
By default, `Switcher` will stack the items vertically. When the container `inline-size` gets bigger than `665px` (converted to `rem`), the items will be displayed horizontally.

<div class="not-content">
	<Switcher>
		<div>
			column 1
		</div>
		<div>
			column 2
		</div>
	</Switcher>
</div>

```jsx
<Switcher>
	<div>
		column 1
	</div>
	<div>
		column 2
	</div>
</Switcher>
```

## More than 2 items

`Switcher` can also be used with more than 2 items. The items then always will be stacked vertically.

<div class="not-content">
	<Switcher>
		<div>
			column 1
		</div>
		<div>
			column 2
		</div>
		<div>
			column 3
		</div>
	</Switcher>
</div>

```jsx
<Switcher>
	<div>
		column 1
	</div>
	<div>
		column 2
	</div>
	<div>
		column 3
	</div>
</Switcher>
```

## Custom properties

```css
/**
 * 1. The threshold at which the items will be displayed horizontally, defaults to `665px` (converted to `rem`).
 * 2. The gap between the items, defaults to `var(--space-6)`.
 * 3. The `align-items` property of the container, defaults to `stretch`.
 */
.your-component-extending-switcher {
	--switcher-threshold: 45ch; /* [1] */
	--switcher-gap: 1rem; /* [2] */
	--switcher-align-items: center; /* [3] */
}
```
