Spacing variables used to let components and their subcomponents breathe with whitespace in relation to one another.

## Example

Adjust a component's spacing by setting the  `--space-[name]` value. In this case we use it for
`margin-block-end`, but it's possible for any property.

```css
.your-component {
  margin-block-end: var(--space-8);
}
```

## One offs
There are some cases where you might want to use a different spacing value for a specific instance. For those instances we've created one offs.

```css
--space-40: #{rem.convert(40 * settings.$baseline)}; // 160px
--space-75: #{rem.convert(75 * settings.$baseline)}; // 300px
--space-100: #{rem.convert(100 * settings.$baseline)}; // 400px
--space-menu-button: var(--space-8);
```
