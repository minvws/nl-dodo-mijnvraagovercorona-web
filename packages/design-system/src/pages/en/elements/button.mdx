A button is used to enable interactivity on your page. By default a button is
rendered as a `button` element. When you pass in the `href` prop it will
convert to a anchor element.

## Variants

### primary

<Button>Button label</Button>

```javascript
<Button>Button label</Button>
```

### secondary

<Button variant="secondary">Button label</Button>

```javascript
<Button variant="secondary">Button label</Button>
```

### tertiary

<Button variant="tertiary">Button label</Button>

```javascript
<Button variant="tertiary">Button label</Button>
```

## Disabled

<Button disabled>Button label</Button>

```javascript
<Button disabled>Button label</Button>
```

## External

When you pass in the `href` prop you can also define if the link is `external`. This adds the external icon. Alternativeley you can also provide a `hiddenLabel` to explain where the link will be opened.

<Button href="#" external hiddenLabel="opens in new tab">
	Button label
</Button>

```javascript
<Button href="#" external hiddenLabel="opens in new tab">
	Button label
</Button>
```

## Custom Icons

You can add an icon to the button.

<Button icon="refresh">Button label</Button>

```javascript
<Button icon="refresh">Button label</Button>
```

### Icon position

By default all icons are aligned `trailing` after the button label.
You can also align the icon before the label by passing in `leading` into `iconPosition`.

<Button icon="refresh" iconPosition="leading">
	Button label
</Button>

```javascript
<Button icon="refresh" iconPosition="leading">
	Button label
</Button>
```

### Icon Animations

You can animate the icon on interaction(`:hover, :focus`).

<Button icon="refresh" interactionIconAnimation="rotate">
	Button label
</Button>

```javascript
<Button icon="refresh" interactionIconAnimation="rotate">
	Button label
</Button>
```
