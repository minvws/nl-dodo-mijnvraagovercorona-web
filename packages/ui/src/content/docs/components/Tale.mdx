import { Tale } from '@lib/components/Tale';

A `Tale` component is used to tell a story with a title and a optional picture.

## Usage

```jsx
import { Tale } from '@dodo/ui/components';

<Tale title="Tale title">
	<p>Some content</p>
</Tale>
```

## Default

<div class="not-content">
	<Tale title="Tale title">
		<p>Some content</p>
	</Tale>
</div>

```jsx
import { Tale } from '@dodo/ui/components';

<Tale title="Tel title">
	<p>Some content</p>
</Tale>
```

## With picture
Providing a [`Picture`](/components/picture) object will display the picture on the side of the tale.

		<p>Interdum et malesuada fames ac ante ipsum primis in faucibus. Integer nec ante neque. Nullam enim urna, vehicula eget nulla vel, luctus elementum diam. Sed molestie aliquam odio, at suscipit odio. Fusce ac gravida diam. Vestibulum aliquam, dolor et euismod ultrices, ex purus dictum lacus, a volutpat ante quam egestas massa.</p>
	</Tale>
</div>

```jsx
import { Tale } from '@dodo/ui/components';

<Tale title="Tel title" picture={{ alt: '', image: { src: 'https://placehold.co/280x280' }}}>
	<p>Some content</p>
</Tale>
```

## Link to tale

You can use the `id` prop to make the tale linkable.

<div class="not-content">
	<Stack>
		<a href="#tale-title">To tale</a>
		<hr />
		<Tale id="tale-title" title="Tale title">
			<p>Interdum et malesuada fames ac ante ipsum primis in faucibus. Integer nec ante neque. Nullam enim urna, vehicula eget nulla vel, luctus elementum diam. Sed molestie aliquam odio, at suscipit odio. Fusce ac gravida diam. Vestibulum aliquam, dolor et euismod ultrices, ex purus dictum lacus, a volutpat ante quam egestas massa.</p>
		</Tale>
	</Stack>
</div>

import { Tale } from '@dodo/ui/components';
