A link is used to show link style elements on the page. By default a Link is
rendered as a `Link` element.

<Link>Link label</Link>

```javascript
<Link>Link label</Link>
```

When you pass in the `href` prop it will
convert to a anchor element.

<Link href="#">Link label</Link>

```javascript
<Link href="#">Link label</Link>
```

## Variants

### tooltip

<Link variant="tooltip">Link label</Link>

```javascript
<Link variant="tooltip">Link label</Link>
```

### restart

<Link variant="restart">Link label</Link>

```javascript
<Link variant="restart">Link label</Link>
```

### restart

<Link variant="back">Link label</Link>

```javascript
<Link variant="back">Link label</Link>
```

## Disabled

<Link disabled>Link label</Link>

```javascript
<Link disabled>Link label</Link>
```

## External

When you pass in the `href` prop you can also define if the link is `external`. This adds the external icon. Alternativeley you can also provide a `hiddenLabel` to explain where the link will be opened.

<Link href="#" external hiddenLabel="opens in new tab">
	Link label
</Link>

```javascript
<Link href="#" external hiddenLabel="opens in new tab">
	Link label
</Link>
```

## Custom Icons

You can add an icon to the Link.

<Link icon="refresh">Link label</Link>

```javascript
<Link icon="refresh">Link label</Link>
```

### Icon position

By default all icons are aligned `trailing` after the Link label.
You can also align the icon before the label by passing in `leading` into `iconPosition`.

<Link icon="external" iconPosition="leading">
	Link label
</Link>

```javascript
<Link icon="external" iconPosition="leading">
	Link label
</Link>
```

### Icon Animations

You can animate the icon on interaction(`:hover, :focus`).

<Link icon="external" interactionIconAnimation="swing">
	Link label
</Link>

```javascript
<Link icon="external" interactionIconAnimation="swing">
	Link label
</Link>
```
