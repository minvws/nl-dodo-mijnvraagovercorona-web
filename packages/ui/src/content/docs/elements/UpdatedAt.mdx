import { UpdatedAt } from '@lib/elements/UpdatedAt';

The `UpdatedAt` component is used to indicate the time and date of the latest content update.

## Usage
Pass in a ISO string in the `time` variable for example `2023-11-29T15:19:35Z`.

```jsx
import { UpdatedAt } from '@dodo/ui/elements';

<UpdatedAt time={'2023-11-29T15:19:35Z'} />
```

<div class="not-content">
    <UpdatedAt time={'2023-08-29T15:19:35Z'} />
</div>
