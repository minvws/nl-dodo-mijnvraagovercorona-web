import { Advice } from '@lib/components/Advice';

The `Advice` component is used to display a piece of advice to the user. Usually this component is placed in the [`components/Hero`](/components/hero) section of the page.

## Usage

```jsx
import { Advice } from '@dodo/ui/components';

<Advice title="Toch ernstige klachten?" subtitle="Neem contact op met je huisarts" image={{src: 'https://placehold.co/32x32'}} />
```
