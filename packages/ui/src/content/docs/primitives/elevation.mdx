`Elevations` are used to offset components from their parents.

## Examples

Adjust the boxshadow of a component with the `box-shadow` property and a `--elevation-frost-[size]` value.

```css
.your-component {
	box-shadow: var(--elevation-frost);
}

.your-component {
	outline: var(--elevation-focusring);
}
```
