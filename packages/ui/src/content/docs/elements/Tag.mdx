import { Tag } from '@dodo/ui/elements';


A `Tag` is a simple component that displays a label and icon for informational purposes. All icons from [`Elements/icon`](/elements/icon) are available.

## Usage

```jsx
import { Tag } from '@dodo/ui/elements';

<Tag icon="appointment">Tag label</Tag>
```
