`Radii` are various kinds of standard border radii. Thes are mapped to our [Spacing](/primitives/spacings) variables.


## Examples

Adjust the border radius of a component with the `border-radius` property and a `--radii-[size]` value.

```css
.your-component {
	border-radius: var(--radii-nano);
}
```
It is also possible to adjust the border radius of an element by adding a utility class `u-radii--[size]`

```html
<div class="u-radii--nano"></div>
```
