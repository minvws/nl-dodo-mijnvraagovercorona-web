import { CardBlock } from '@lib/components/Card';

The `CardBlock` component is used to display a card with header, content and (optional) button(s).

## Usage

```jsx
import { CardBlock } from '@dodo/ui/components';

<CardBlock title="[title]">[content]</CardBlock>
```


## Example

<div class="not-content">
	<CardBlock title="This is a title">
		<p>At vero eos et accusamus et iusto odio dignissimos ducimus qui blanditiis praesentium voluptatum deleniti atque corrupti quos dolores et quas molestias excepturi sint occaecati cupiditate non provident</p>
	</CardBlock>
</div>

```jsx
	<CardBlock title="This is a title">
		<p>Some content</p>
	</CardBlock>
```
