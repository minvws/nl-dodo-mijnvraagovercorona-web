`Chapeaus` are words or short phrases that precedes a heading.

## Usage

```jsx
import { Chapeau } from '@dodo/ui/elements';

<Chapeau>This is a Chapeau title</Chapeau>
```

## Example

<div class="not-content">
    <Chapeau>This is a Chapeau title</Chapeau>
    <Heading level={1}>This is a heading</Heading>
</div>
