The `Card` component is a basic card element with some optional layout variants. 
The component kan be used to wrap around content or other components such as `CardAdvice`, `CardSupport`, `CardDay`, `CardPrivacy`

```jsx
import { Card } from '@mvoc/ui/components';

<Card> //content </Card>
<Card variant={'filled'}> // filled variant card </Card>
```
