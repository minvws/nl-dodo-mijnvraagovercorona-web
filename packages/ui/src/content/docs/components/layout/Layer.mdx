import { Layer } from '@lib/components/Layout/Layer';

The `Layer` component is used to create a layer of content that can be positioned anywhere on the page. It is useful for creating distinct sections of content.

## Usage

```jsx
import { Layer } from '@dodo/ui/components';

<Layer>
	// content
</Layer>
```

## Default

By default, the `Layer` component will render a `section` with a white background.

<div class="not-content">
	<Layer>
		<p>Lorem ipsum dolor sit amet consectetur adipisicing elit. Perferendis est voluptate deleniti atque molestiae, consectetur temporibus labore autem illo sequi dolorum veniam et voluptatem distinctio numquam perspiciatis eum iste! Quo?</p>
	</Layer>
</div>

```jsx
<Layer>
	// content
</Layer>
```

## Render as another element

By default, the `Layer` component will render a `section`. You can change this by passing the `as` prop.

<div class="not-content">
	<Layer>
		<p>Default layer, rendering a section</p>
	</Layer>
	<Layer as="div">
		<p>rendering a div</p>
	</Layer>
	<Layer as="footer">
		<p>Rendering a footer</p>
	</Layer>
</div>

```jsx
<Layer>
	// content
</Layer>
<Layer as="div">
	// content
</Layer>
<Layer as="footer">
	// content
</Layer>
```

## Custom properties

The following custom properties are available on `Layer`:

```scss
/**
 * 1. The padding of the layer on the `block-axis`, defaults to `var(--density-layer)`.
 * 2. The padding of the layer on the start of the `block-axis`, defaults to `var(--layer-padding)`.
 * 3. The padding of the layer on the end of the `block-axis`, defaults to `var(--layer-padding)`.
 */
.your-component-extending-layer {
	--layer-padding: 1rem; /* [1] */
	--layer-padding-block-start: 1rem; /* [2] */
	--layer-padding-block-end: 1rem; /* [3] */
}
```
