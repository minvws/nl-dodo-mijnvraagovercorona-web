title: "Breadcrumbs"

import { Breadcrumbs } from '@lib/components/Breadcrumbs';

The `Breadcrumbs` component is used to display the current location within a hierarchy of pages. It is used to help users navigate between pages and understand where they are within the site.
On smaller screens we'll only show the previous page. On larger screens we'll show all the pages.

```jsx
import { Breadcrumbs } from '@dodo/ui/components';

<Breadcrumbs items={[
	{slug: 'item-1', title: 'Item 1'},
	{slug: 'item-2', title: 'Item 2'},
	{slug: 'item-3', title: 'Item 3'},
	{slug: 'item-4', title: 'Item 4'}
]} />
```
