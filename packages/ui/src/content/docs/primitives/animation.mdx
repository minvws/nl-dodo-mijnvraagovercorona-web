## Interaction (`:hover`, `:focus`) animations

We have some predefined animation utility classes that can be used in your interactive components like `Button` & `Link`:

```css
.u-animation-interaction--swing {
	&:hover,
	&:focus {
		--u-animation-subscribe: var(--animation-swing);
	}
}

.u-animation-interaction--bounce {
	&:hover,
	&:focus {
		--u-animation-subscribe: var(--animation-bounce);
	}
}

.u-animation-interaction--rotate {
	&:hover,
	&:focus {
		--u-animation-subscribe: var(--animation-rotate);
	}
}

.u-animation-interaction--rotate-reverse {
	&:hover,
	&:focus {
		--u-animation-subscribe: var(--animation-rotate-reverse);
	}
}
```

and can be used in your components like this:
First you need to "subscribe" a (sub-)element to the animation by setting the `--u-animation-subscribe` custom property.

```css
.your-component .e-icon {
	animation: var(--u-animation-subscribe);
}
```
...and then you can add the animation to your component:

```jsx
<div className="your-component  u-animation-interaction--[ANIMATION-NAME]"><Icon /></div>
```

Alternatively you can use the animation custom properties directly in your components:

```css
.your-component {
	animation: var(--animation-bounce);
}
```

## Components using animations
In the `Button` and  `Link` components already have support for these animations. You can add animations to the icon by providing the `interactionIconAnimation` prop.

```jsx
<Button icon="arrow-right" interactionIconAnimation="swing">button label</Button>
```
