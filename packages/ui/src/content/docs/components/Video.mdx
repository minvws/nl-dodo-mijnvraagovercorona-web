import { Video } from '@lib/components/Video';

The `Video` component is used to link to a Vimeo or Youtube video.

## Usage

```jsx
import { Video } from '@mvoc/ui/components';

<Video title="Video title" url="https://www.youtube.com/watch?v=pbD0Bqgegfk" image={{ src: 'https://placehold.co/640x360' }}/>
```
