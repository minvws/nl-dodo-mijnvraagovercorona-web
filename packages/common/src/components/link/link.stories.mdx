# Link

The link component can be used to either link via the router inside the app, or link to an external website.
By default the component is prefixed with a chevron icon â€” which can be disabled via the `withChrevron={false}` prop.

## Props

The `as` and `onClick` prop are only required if you provide `as="button"`. Storybook however shows all of them in one table. As soon as you use
`as="button"` the `href` and `external` props aren't required anymore.
