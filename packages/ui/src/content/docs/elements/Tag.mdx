import { Tag } from '@mvoc/ui/elements';


A `Tag` is a simple component that displays a label and icon for informational purposes.

## Usage

```jsx
import { Tag } from '@mvoc/ui/elements';

<Tag icon="appointment">Tag label</Tag>
```
