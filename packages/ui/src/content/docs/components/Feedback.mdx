The Feedback component is used to gain information from the user in form of like & dislike buttons. When the user selects one option there is a foldout with a link to an external survey tool.

import { Feedback } from '@lib/components/Feedback';


## Usage

```jsx
import { Feedback } from '@dodo/ui/components';

<Feedback url="#" name="Feedback formulier" />
```

## Default

<div class="not-content">
	<Feedback url="#" name="Feedback formulier" />
</div>
