import { Advice } from '@lib/components/Advice';

The `Advice` component is used to display a piece of advice to the user.

## Usage

```jsx
import { Advice } from '@mvoc/ui/components';

<Advice title="Toch ernstige klachten?" subtitle="Neem contact op met je huisarts" image={{src: 'https://placehold.co/32x32'}} />
```
