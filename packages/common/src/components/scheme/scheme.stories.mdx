# SchemeBlock And SchemeBullet

The `SchemeBlock` and `SchemeBullet` components are used for rendering quarantaine and travel plans for the users.
Each `SchemeBlock` has a title, subtitle and a possible day label, showing 1 specific moment in time. Nested inside each
`SchemeBlock` can be one or multiple `SchemeBullet` components. Each of these bullets can hold WYSYWIG content (including paragraphs).
