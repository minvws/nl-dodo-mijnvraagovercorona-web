import { CardSupport } from '@lib/components/Card';
import { Button } from '@lib/elements/Button';

The `CardSupport` component is used to display a card with an image in the card header, content and (optional) button(s).

## Usage
Title, image props can be used to add title and image.

```jsx
import { CardSupport } from '@dodo/ui/components';

<CardSupport title="[insert title]" image="[insert image source]"> //content </CardSupport>
```


## Example

<div class="not-content">
	<CardSupport title="This is a title" image={{src: 'https://placehold.co/44x44'}}>At vero eos et accusamus et iusto odio
	dignissimos ducimus qui blanditiis praesentium voluptatum deleniti atque corrupti quos dolores et quas molestias excepturi
	sint occaecati cupiditate non provident
	<Button variant="secondary">With button</Button></CardSupport>
</div>
```jsx
	<CardSupport title="This is a title" image={{src: 'https://placehold.co/44x44'}}> Some content
	<Button variant="secondary">With button</Button></CardSupport>
```
